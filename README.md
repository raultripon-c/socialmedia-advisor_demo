# Social Media Advisor Demo

Standalone demo of the **Social Media Advisor** flows extracted from `txe-ui`.

**Live preview:** [https://raultripon-c.github.io/socialmedia-advisor_demo/](https://raultripon-c.github.io/socialmedia-advisor_demo/)

## Run

```bash
npm install
npm run dev
```

Opens at [http://localhost:4310](http://localhost:4310) on the Campaigns home screen.

## What’s interactive

Everything inside Social Media Advisor:

- Campaigns list / generate flow / templates / nudges
- Content Board
- Testimonial / Video Hub campaign creation
- Post preview workspace

## What’s not

The Phenom shell (sidebar links other than Social Media Advisor, avatar, search) is visual only for the demo — clicks are disabled.
